API Documentation:

Here are two ways that you can access our API.

Through CURL Request:
curl -X "GET" https://carpricestool.eu.pythonanywhere.com/api/cars/results
-H "Content-Type: application/json"
-H "Authorization: Token YOURTOKEN"
-d 'filter1=value1'
-d 'filter2=value2'
-d 'filter3=value3'

Through URL Request:
https://carpricestool.eu.pythonanywhere.com/api/cars/results?auth_token=YOURTOKEN&filter1=value1&filter2=value2&filter3=value3

Following filters are available:
Filter Values or Examples Requirements
make eq. 'Kia'
state Used, New (if you don't provide this filter, both will be returned)
model eq. 'Ceed'
offer_type business, person
mileage_less_more mileage_less_than, mileage_more_than Needs mileage
mileage eq. '125000' Needs mileage_less_more
production_year_less_more production_year_less_than, production_year_more_than, production_year_exact Needs production_year
production_year eq. '2018' Needs production_year_less_more
price_less_more price_less_than, price_more_than Needs price and price_currency
price eq. '12000' Needs price_less_more and price_currency
price_currency USD, PLN, EUR Needs price_less_more and price
engine_capacity_less_more engine_capacity_less_than, engine_capacity_more_than, engine_capacity_equal Needs engine_capacity
engine_capacity eq. '1.8' Needs engine_capacity_less_more
engine_power_less_more engine_power_less_than, engine_power_more_than, engine_power_equal Needs engine_power
engine_power eq. '90' Needs engine_power_less_more


Please note that API will return only first 500 cars. If your search is hitting this limit, please narrow it.
